## 🚗 CarFinder - Plataforma de Busca de Veículos
O CarFinder é uma aplicação web moderna desenvolvida para facilitar a busca e visualização de veículos. Este projeto foi desenvolvido como parte prática do curso de Programação Frontend com React e Javascript do IFSC Canoinhas.

O objetivo principal foi aplicar conceitos avançados de arquitetura de componentes, gerenciamento de estado e integração com serviços de backend e banco de dados.

## 🚀 Tecnologias Utilizadas
Esta aplicação utiliza a Stack MERN e outras ferramentas modernas:

Frontend: React.js com Vite (para maior performance no desenvolvimento).

Gerenciamento de Estado: React Hooks e Context API.

Roteamento: React Router para navegação entre páginas.

Estilização: Autoprefixer e CSS moderno.

Backend & Integração: Conhecimentos em Node.js, Express e integração com Firebase.

Banco de Dados: Noções de MongoDB (Stack MERN).

## 🔧 Como Executar o Projeto

1. Clone o repositório:
   ```bash
   git clone [https://github.com/seu-usuario/carfinder.git](https://github.com/seu-usuario/carfinder.git)

2. Instale as dependências:

No terminal:

3. Inicie o servidor de desenvolvimento:

No terminal:
npm run dev
